Smart Calculator.

Smart Calculator - удобный инструмент сложных вычислений.

Дополнительные функции - кредитный калькулятор и депозитный калькулятор.

Установить калькулятор: make install
Удалить калькулятор: make uninstall
Тесты: make tests
Вернуть к состоянию репозитория: make clean
